# Трекер изучения технологий (Technology Tracker)

React-приложение для отслеживания прогресса в изучении современных технологий.

функционал

- **Карточки технологий:** Отображение технологий с названием, описанием и статусом
- **Три статуса:** 
  -  Изучено (completed)
  -  В процессе (in-progress)
  -  Не начато (not-started)
- **Панель прогресса:** Общая статистика по всем технологиям
- **Адаптивный дизайн:** Работает на всех устройствах
- **Анимации:** Плавные переходы и hover-эффекты

Задействованные технологии

- React 18
- Vite (сборщик)
- CSS Modules (стилизация)
- GitHub Pages (деплой)
